## Philosophy

> *"An idea without execution is just a thought. I'd rather ship something imperfect than perfect something that never ships."*

- I believe the best engineers are product thinkers first, and technologists second
- AI is not replacing engineers — it's giving solo builders the leverage of a whole team
- Every system I build, I ask: *what happens when this gets 10x the load?*
- Code is a means, not an end — the real job is solving problems people actually have
